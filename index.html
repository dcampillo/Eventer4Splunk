Eventer4splunk

Eventer4Splunk is a c# log writer for Splunk TM

Download .zip Download .tar.gz View on GitHub

Eventer4Splunk is a lightweight c# library aimed to ease writing logs in the Splunk (http://www.splunk.com) format.

Features:

  • Generate Splunk formatted logs
  • Support all version of Splunk
  • Easily send logs to a TCP/UDP Data Input
  • Factory based event generator
  • Memory queue for faster generation
  • Support correlated event aka. Transaction
